# **App Name**: Care Billing App

## Core Features:

- Mobile Invoice Creation: Streamlined single-column layout for quick invoice creation on mobile devices, optimized for thumb usage.
- POS-Style Tablet Interface: Tablet UI mimicking a Point of Sale system with product/project list and quick select on the left, and selected items with quantity controls on the right.
- Dynamic Layout Rendering: Conditionally render different UI components based on screen size to achieve a responsive layout for mobile and tablet.
- Invoice Sharing: Share generated invoices quickly via WhatsApp from both mobile and tablet interfaces.
- Data Persistence: Store and retrieve invoice data using Google Sheets as the backend, ensuring data consistency across devices.
- Backend Calculations: Leverage backend logic in Google Sheets to perform real-time calculations of subtotal, tax, discount, and total amounts.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey trust and efficiency in financial transactions.
- Background color: A light blue (#E1F5FE) to provide a clean and calming backdrop, ensuring readability.
- Accent color: A contrasting orange (#FF9800) for key interactive elements such as 'Generate Invoice' and 'Pay,' drawing attention and signaling action.
- Font: 'PT Sans', sans-serif for clarity, modernity and good readability on a range of devices.
- Mobile: Single-column layout for easy thumb navigation.
- Tablet: POS-style layout with product list on the left and invoice details on the right.
- Ensure all important interactive elements are positioned and sized so as to avoid accidental clicks.
